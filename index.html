How it works

Three steps. That's it.

From enrollment to verification in a single API call.

1

Enroll

Send a photo + user ID to /enroll. We generate and store a face embedding securely.

2

Verify or Identify

Send a live photo to /verify or /identify. We match it against your database.

3

Get a Result

Receive a JSON response with verified: true/false and a confidence score in milliseconds.
