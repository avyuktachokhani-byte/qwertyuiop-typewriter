Features

Everything you need to verify identity.

A complete biometric terminal that runs entirely in the browser.

🖥️

Zero server required

Everything runs client-side. Face detection, descriptor generation, and matching all happen in the browser. No backend, no hosting costs, no latency.

👁️

Real-time face tracking

Live camera overlay detects and tracks faces with bounding boxes in real time using TinyFaceDetector.

🔀

Two verification modes

Identity mode finds who someone is. Verify mode confirms they are who they claim to be.

⚙️

Built-in admin panel

Enroll users with camera or image upload, manage the database, and view audit logs — all in one interface.

🧠

128-dimensional face descriptors

Each face is converted to a mathematical fingerprint. Matching uses cosine distance with a configurable 0.55 threshold for high accuracy.

const det = await faceapi
  .detectSingleFace(video)
  .withFaceLandmarks()
  .withFaceDescriptor();

const match = findMatch(det.descriptor);
// → { memberId: "1234567", name: "Rashi", confidence: 98.2 }
🔒

Privacy-first

No raw images stored. Biometric data stays in the user's browser. Full audit trail for every scan.

How it works

Three steps. That's it.

From enrollment to verification in a single interface.

1

Enroll

Open the admin panel, enter a Member ID and name, then capture a photo via camera or upload. A face descriptor is generated and stored locally.

2

Scan

Switch to the terminal view. The camera tracks faces in real time. Hit Run Verification to compare against the enrolled database.

3

Get a Result

See an authorized or denied result with the matched user's name, ID, and confidence score. Every scan is logged for audit.
